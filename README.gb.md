## Getting Started

### Dependencies

* Nodejs, npm or yarn
* Reactjs
* Firebase
### Installing

* ```
  git clone https://github.com/phamquyetthang/quoc-am-client.git 
  ```
* ```
  yarn install
  ```
* You alse use npm to install project
  
### Executing program

* ```
  yarn start
  ```
